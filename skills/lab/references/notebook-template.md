# 実験ノート・論文要約テンプレート

---

## 実験ノート（experiments/notebook/EXP-XXX.md）

実験家が実験完了時に作成する。成功・失敗を問わず記録する。
追記のみ。消さない・編集しない。

```markdown
# EXP-XXX: [実験名]
日時: [YYYY-MM-DD HH:MM]
関連仮説: [Hn]

## 目的
[この実験で何を確かめるか]

## 設定
- ハイパーパラメータ:
- seed:
- 環境:
- コミット:
- wandb:

## ベースライン
[何と比較するか。過去実験 EXP-XXX がある場合はそれも参照]

## 結果
[数値 ± 標準偏差、n=回数、ベースラインとの差分]

## 所見
[結果から何が言えるか]

## 失敗・異常（該当時のみ）
- エラー内容:
- 再現手順:
- 原因仮説:
```

### 命名規則

- ファイル名: `EXP-001.md`, `EXP-002.md`, ...（連番）
- 既存の最大番号を確認し、次の番号を使う
- 番号は3桁ゼロ埋め

---

## 論文要約（literature/active/[著者名year]-summary.md）

文献調査員が論文要約時に作成する。1論文1ファイル。

```markdown
# [著者名year]: [論文タイトル]
一言: [この論文を一言でいうと]

## 目的
[この研究は何を解決しようとしているか]

## 手法
[どのようなアプローチを取っているか]

## 結果
[主要な結果]

## 限界
[著者が認めている限界、または読み手が気づいた限界]

## 自分たちの研究との関連
- 類似点:
- 差異:

## 重要度
必読 / 参考程度 / 関連薄い

## BibTeX
```bibtex
@article{著者名year,
  ...
}
```
```

### 命名規則

- ファイル名: `[第一著者の姓][年]-summary.md`（例: `chen2025-summary.md`）
- kebab-case

## BibTeX（literature/active/known_papers.bib）

文献調査員が管理する。タグ付き。

```bibtex
% タグ: 読了=read, 未読=unread, 斜め読み=skimmed
% 例: keywords = {read, important}

@article{example2025,
  author = {},
  title = {},
  journal = {},
  year = {2025},
  keywords = {unread}
}
```
